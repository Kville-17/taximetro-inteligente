# IA Contextual

Sistema de IA especializado en información relacionada con tarifas, regulaciones y funcionamiento del sistema.

## Objetivos

* Explicar cobros.
* Responder dudas.
* Detectar anomalías.
* Consultar estadísticas.
* Mejorar transparencia.

## Importante

La IA no funcionaría como chatbot general, sino como asistente contextual especializado.
